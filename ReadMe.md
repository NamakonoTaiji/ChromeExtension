## CopilotExtension概要

文字を選択してショートカットキーを押すとポップアップにCopilotの回答が出てくるようにする拡張機能を制作中です。<br>
https://copilot.microsoft.com にアクセスし、Copilotのタブを開いたら回答させたい内容のテキストを選択、ショートカットキー（Ctrl+Shift+Y）を押すとCopilotに文章を送ることができます。<br>
コードの詳細はbackground.jsでコメントアウトしました。<br>
問題によりポップアップ機能を実装できていません。<br>

## 使い方

当拡張機能をインストール、解凍します。<br>
chromeを開き右上のドロップダウンから拡張機能 > 拡張機能を管理 > 右上にあるデベロッパーモードを有効 > パッケージ化されていない拡張機能を読み込むを選択、ChromeExtension-masterフォルダを選択します。<br>

タブにCopilotがある状態で文字を選択、Ctrl+Shift+Yを押すと自動でCopilotのテキストエリアに選択した文字を貼り付け、送信を行います。<br>
